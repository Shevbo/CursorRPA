# Shectory Welcome Frame Template

Шаблон каркаса welcome/login страницы для прикладных проектов.

Состав:
- `welcome-frame.html` — базовая разметка экрана
- `welcome-frame.css` — базовые стили

Что обязательно сохранить при кастомизации:
- логотип Shectory слева сверху,
- логотип проекта + версии модулей справа сверху,
- единая область логина,
- большой информационный фрейм (может быть проект-специфичным).

Ссылка на стандарт:
- `docs/welcome-page-standard-ru.md`
